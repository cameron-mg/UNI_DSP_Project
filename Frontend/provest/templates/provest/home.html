How It Works

1. The Property Search tab at the top will be your main way to interact with the system.
2. You need to provide a postcode in the space provided on the search tab.
3. After clicking "search" the system will find properties in the search radius currently on the market.
4. The system will present the properties to you ordered by possible return on investment.
5. More details about the properties can be found by clicking on the "view details" button on each property returned.

How We Predict Future House Prices

- The machine learning model has been trained on all uk property transactions from 1995 until 2017.
- This data allows the model to predict the future price of a house based on various metrics such as postcode, property type and date of transfer.
- The model has been developed to a suitable degree of accuracy however can make mistakes and all predictions are only possibilities.

Real-Time Data

- The real-time property data is provided by PropertyDataUK.
- All properties found from the search will have a sale page accessible through a link displayed with the properties details.
